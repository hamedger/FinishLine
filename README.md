# Finish Line Oil Change and Repair

SEO-focused website for **Finish Line Oil Change and Repair** — a family-owned auto shop in Livonia, MI.

## Features

- Hero section with call & directions CTAs
- Full services menu (Jiffy Lube inspired)
- Monthly coupons with auto-expiring end-of-month dates
- Scrolling customer testimonials
- Fleet management section with quote request form
- Tips from Finish Line (SEO content blocks)
- Contact form + store hours + embedded map
- Sticky mobile call bar
- LocalBusiness JSON-LD schema for SEO

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Works on [Vercel](https://vercel.com), [Netlify](https://netlify.com), or any Node.js host:

```bash
npm run build
npm start
```

Point `finishlineoil.com` to your deployment.
